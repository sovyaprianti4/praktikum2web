# Framework Lanjutan ( CRUD )
## Deskripsi
Project ini merupakan aplikasi web sederhana berbasis CodeIgniter 4 yang digunakan untuk mengelola data artikel.  
Aplikasi ini memiliki fitur CRUD (Create, Read, Update, Delete) serta tampilan yang sudah dibuat lebih modern dan estetik.
